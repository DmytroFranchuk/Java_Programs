- Проект является веб-приложением, разработанным с использованием фреймворка Spring Boot в учебных целях. 
- Функциональность приложения:
- приложение bankApp работает с базой данных MySQL под названием bankdb
- в базе создаются с помощью инструмента Liquibase таблицы managers, clients, products, accounts, agreements, transactions
- в файлах инициализации прописаны внешние ключи для обеспечения целостности данных и связей между таблицами это гарантия, того что данные, 
  хранящиеся в таблицах, остаются согласованными и соответствуют правилам целостности
- таблицы так же проиндексированы для оптимизации производительности запросов к базе
