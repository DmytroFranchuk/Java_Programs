Доменная модель погодного сервиса в телеграмм-боте:
Основные сущности:
- Пользователь
    - идентификатор пользователя в Telegram
- Погодный запрос
    - местоположение (город или координаты)
    - дата запроса
    - время запроса
- Погодные данные
    - температура
    - влажность
    - скорость ветра
    - осадки
    - метеорологические параметры
- Погодный источник
    - название портала
    - API ключ

Процессы взаимодействия:
    Пользовательский запрос:
        Действие пользователя:
            Отправка сообщения с запросом на погоду
        Входные данные:
            Текстовое сообщение с указанием города или координат и даты/времени
        Шаги выполнения:
            Извлечение местоположения и времени из запроса
            Создание объекта "Погодный запрос"
            Отправка запроса внешнему погодному источнику
    Запрос к погодному источнику:
        Действие программы:
            Отправка запроса к внешнему погодному порталу для получения данных
        Входные данные:
            Местоположение и время из "Погодного запроса"
        Шаги выполнения:
            Использование API внешнего портала для получения погодных данных
            Создание объекта "Погодные данные"
    Ответ пользователю:
        Действие программы:
            Отправка сообщения с погодной информацией пользователю
        Входные данные:
            Объект "Погодные данные"
        Шаги выполнения:
            Форматирование текста с погодной информацией.
            Отправка сообщения пользователю в Telegram.

Важные поля для сущностей:
    Пользователь:
        Идентификатор Telegram
    Погодный запрос:
        Местоположение (город или координаты)
        Дата и время запроса
    Погодные данные:
        Температура
        Влажность
        Скорость ветра
        Осадки и другие метеорологические параметры
    Погодный источник:
        Название портала
        API ключ

Use Cases:
    Запрос погоды по городу:
        Пользователь:
            Какая погода в Берлине сегодня
        Программа:
            Извлекает местоположение и текущую дату из запроса, отправляет запрос к погодному источнику,
            форматирует и отправляет ответ пользователю
    Запрос погоды по координатам:
        Пользователь:
            Какая погода на координатах 25.7558, 47.6176 завтра в 15:00
        Программа:
            Извлекает координаты и указанное время из запроса, отправляет запрос к погодному источнику,
            форматирует и отправляет ответ пользователю
    Некорректный запрос:
        Пользователь:
            Дай погоду
        Программа:
            Обнаруживает отсутствие необходимой информации в запросе и уведомляет пользователя о
            необходимости предоставить более конкретные данные
    Приветствие:
        Пользователь:
            Привет
        Программа:
            Реагирует на приветствие пользователя, предлагает помощь и инструкции по использованию бота
    Отмена запроса:
        Пользователь:
            Отмена
        Программа:
            Отменяет текущий запрос и сообщает пользователю о завершении операции
